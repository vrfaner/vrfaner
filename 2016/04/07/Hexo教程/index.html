HEXO你的私人博客（一）

听朋友说HEXO是个很不错的东西，我看了一下，感觉这个逼格极高的程序猿写作方式。所以准备做点什么，于是就了解了一下Hexo

hexo出自台湾大学生tommy351之手，是一个基于Node.js的静态博客程序，其编译上百篇文字只需要几秒。hexo生成的静态网页可以直接放到GitHub Pages，BAE，SAE等平台上。Hexo颯爽登場！

  • 如果你对默认配置满意，只需几个命令便可秒搭一个hexo。
  • 如果你跟我一样喜欢折腾下，30分钟也足够个性化。
  • 如果你过于喜欢折腾，可以折腾个把星期，尽情的玩。

搭建的过程可能会比其他类型的博客网站繁琐一点，但一旦搭建完成，写文章就会及其简单。

仅仅需要2行命令，就可以完成全部工作。

$ hexo g

$ hexo d

搞定！

废话不多说，进入正题。

环境准备(windows)

  • Node.js
  • Git
  • MarkPad

GitHub

  1. 创建GitHub 账号
  2. 创建你的仓库，改仓库名称必须与你账户名称对应『your_user_name.github.io』，切记！

恭喜你准备工作基本完成！接下来只需要使用 npm 即可完成 Hexo 的安装
Git 安装完成后 鼠标右键菜单栏会多出2个菜单

点击 Git bash here

$ npm install -g hexo-cli

安装hexo完成后，对hexo初始化

$ hexo init <folder>

安装hexo相关依赖

$ npm install

cd 到你的init目录，执行如下命令，生成静态页面至hexo\public\目录。

$ cd <folder>

$ hexo generate

执行如下命令，启动本地服务， 预览页面

$ hexo server

浏览器输入http://localhost:4000就可以看到效果。

尽量使用高版本浏览器，有些主题不支持低版本浏览器

OK！
